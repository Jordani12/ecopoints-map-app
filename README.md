# ♻️ Ecopoints Map — Santos, SP

> *An interactive map of Ecopoints (waste collection points) in the city of Santos, SP — Brazil.*
> *Um mapa interativo dos Ecopontos da cidade de Santos, SP — Brasil.*

---

## 🇺🇸 English

### 📖 About
**Ecopoints Map** is a web application that displays all official waste collection points (Ecopontos) in Santos, SP. Developed as an Extension Project for **UNIP**, the goal is to make it easier for citizens to find the nearest ecopoint and what materials each one accepts.

### ✨ Features
- 🗺️ Interactive map powered by **Leaflet.js**
- 📍 Geolocation — find ecopoints near you
- 🔍 Search by name or neighborhood
- 🏷️ Filter by accepted material type
- 📋 Side panel with detailed information
- 💬 Popups with ecopoint details on marker click

## 🇧🇷 Português

### 📖 Sobre
**Mapa de Ecopontos** é uma aplicação web que exibe todos os pontos de coleta de resíduos (Ecopontos) oficiais da cidade de Santos, SP. Desenvolvido como Projeto de Extensão para a **UNIP**, com o objetivo de facilitar o acesso dos cidadãos ao ecoponto mais próximo e saber quais materiais cada um aceita.

### ✨ Funcionalidades
- 🗺️ Mapa interativo com **Leaflet.js**
- 📍 Geolocalização — encontre ecopontos perto de você
- 🔍 Busca por nome ou bairro
- 🏷️ Filtro por tipo de material aceito
- 📋 Painel lateral com informações detalhadas
- 💬 Popups com detalhes do ecoponto ao clicar no marcador
